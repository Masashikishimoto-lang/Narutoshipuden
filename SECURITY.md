# セキュリティポリシー

## 対応バージョン

本リポジトリは単一の静的サイト（HTML/CSS/JS）です。常に `main` ブランチの最新版のみをサポートします。

| バージョン | サポート対象 |
| --- | --- |
| main（最新） | ✅ |
| それ以前 | ❌ |

## 脆弱性の報告

本サイトはサーバーサイド処理やユーザーデータの収集を行わない静的ページですが、XSS等のフロントエンド脆弱性を発見した場合は以下の方法でご報告ください。

- リポジトリの **Issues** から `[SECURITY]` を件名に付けて報告する
- 機密性の高い内容の場合は、公開Issueを避け、リポジトリオーナーへ直接ご連絡ください

報告後、可能な限り速やかに調査・対応します。
