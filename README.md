# NARUTO -ナルト- 疾風伝 資料アーカイブ

非公式・教育目的のファンアーカイブページ。NARUTO -ナルト- 疾風伝の物語構成、登場人物、放送年表、制作情報、文化的影響などを一般公開情報からまとめています。

## 📌 このリポジトリについて

[#このリポジトリについて](#-このリポジトリについて)

このプロジェクトは単一の静的HTMLページとして構築されており、以下の内容を含みます。

| セクション | 内容 |
|---|---|
| 概要 (Synopsis) | 作品全体のあらすじ紹介 |
| 章立て (Arcs) | 物語構成のまとめ |
| 登場人物 (Characters) | キャラクター紹介 |
| 相関図 (Relations) | キャラクター相関図 |
| 声優・キャスト (Cast) | 声優キャスト情報 |
| 年表 (Timeline) | 放送年表 |
| 制作 (Production) | 制作スタジオ・スタッフ情報 |
| 主題歌 (Themes) | オープニング・エンディング曲一覧 |
| 劇場版 (Movies) | 関連映画作品 |
| 用語集 (Glossary) | 作品内用語の解説 |
| 五大国 (Nations) | 忍の里・国の設定紹介 |
| 番外編 (Bonus) | 補足コンテンツ |
| 影響 (Impact) | 文化的影響のまとめ |
| FAQ | よくある質問 |
| 視聴先 (Whitelist) | 正規配信プラットフォーム一覧 |

> **注記：** 本サイトは集英社／スタジオぴえろ／テレビ東京など公式とは一切関係のないファン制作の非公式アーカイブです。全ての商標・著作権は各権利者に帰属します。

## 🚀 使い方

[#使い方](#-使い方)

### ローカルで開く

```bash
git clone https://github.com/<username>/<repo-name>.git
cd <repo-name>
open index.html   # macOS
# または、ブラウザで index.html をダブルクリック
```

### GitHub Pages で公開する

1. リポジトリの **Settings → Pages** を開く
2. **Source** を `main` ブランチ / `/ (root)` に設定
3. 数分後、`https://<username>.github.io/<repo-name>/` で公開されます
