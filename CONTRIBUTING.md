# コントリビュートガイド

このプロジェクトへの貢献に興味を持っていただきありがとうございます！

## 貢献できること

- 🐛 誤字・脱字・情報の誤りの修正
- ➕ 新しいセクションや情報の追加（放送情報、キャスト情報の更新など）
- 🎨 デザイン・レイアウトの改善
- ♿ アクセシビリティの改善
- 🌍 翻訳（他言語版の追加）

## 手順

1. このリポジトリを **Fork** する
2. 作業用ブランチを作成する
   ```bash
   git checkout -b fix/nama-perubahan
   ```
3. 変更を加えてコミットする
   ```bash
   git commit -m "fix: 説明を修正"
   ```
4. Fork したリポジトリに push する
   ```bash
   git push origin fix/nama-perubahan
   ```
5. **Pull Request** を作成する

## ルール

- 内容は必ず一般公開情報（Wikipedia、公式サイト、ニュース記事など）に基づいてください。
- 著作権で保護された画像・文章をそのまま転載しないでください。
- キャラクター・作品名などの固有名詞の表記揺れに注意してください。
- 誹謗中傷やネタバレの取り扱いには配慮してください。

## Issue の報告

バグ報告や情報の誤りは [Issues](../../issues) からお願いします。テンプレートに沿って記入してください。

ご協力ありがとうございます！🙏
